🚀 Como configurar o Banco de Dados?

  1. Crie uma Planilha Google: Crie uma planilha em branco. Opcionalmente, adicione na primeira linha os cabeçalhos: ID na coluna A, Musica na B e Likes na C.
  2. Abra o Apps Script: Na planilha, clique em Extensões > Apps Script.
  3. Cole o código do Backend: Substitua todo o conteúdo do editor pelo código presente no arquivo apps-script/codigo.gs deste projeto.
  4. Implante como Web App: Clique em Implantar > Nova implantação. Escolha App da Web (na engrenagem).
  5. Defina as permissões de acesso: Configure "Executar como" para Você e "Quem tem acesso" para Qualquer pessoa.
  6. Obtenha o URL da API: Conclua a implantação, copie a URL do App da Web gerada e cole no formulário de geração ao lado.
💡 O Google Sheets funciona como banco de dados em tempo real e gratuito!
