## Frase descritiva: Adicionando botão de cópia no html

Este changelog registra a modificação feita no arquivo HTML do painel do gerador (`generator/public/index.html`) para acrescentar o botão de interação na barra lateral de instruções.

### Prompt Motivador
> Pedido do usuário para colocar um botão para copiar o código do Google Apps Script (.gs) diretamente da interface.

- **Agora**: Adicionou-se o botão `#btn-copy-gs` com o texto "Copiar Código (.gs)" localizado estrategicamente abaixo das etapas de implantação da planilha.
